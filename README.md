# BrandGrowth Media
A modern, data-obsessed growth engineering firm website. 

## Overview
BrandGrowth Media provides measurable and scalable growth systems for ambitious brands. The platform is built with a focus on high-performance web development, AI automation, and paid acquisition marketing (Google, Facebook, Instagram Ads, Local SEO, etc.).

## Features
- **Responsive Layout**: Engineered for all screen sizes, including targeted mobile breakpoints.
- **Service Specific Call-To-Actions**: Highly detailed individual service pages with cross-navigation.
- **Custom UI Components**: Interactive 3D CSS visualizations and modern card hover effects.
- **Optimized Assets**: Lightweight CSS architecture structured via a custom `design-system.css`.
- **Dynamic CSS Components**: Standalone structural definitions inside `components.css`.

## Development setup
To run this project locally, start a simple HTTP server in the root directory:

```bash
python3 -m http.server 8080
```

Navigate to `http://localhost:8080` to view the website.

## Tech Stack
- HTML5
- Vanilla CSS3
- Vanilla JavaScript
